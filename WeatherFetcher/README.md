# Weather CLI Tool
A simple Python automation script that fetches real-time weather data from the OpenWeatherMap API using a clean command-line interface.
It loads API keys securely from a .env file, handles errors gracefully, and logs each weather query to a CSV file for tracking.

## Features
1. Fetch weather for any city via CLI
2. Secure API key handling with python-dotenv
3. CSV logging for automation workflows
4. Error handling and clean output
